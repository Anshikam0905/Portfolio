**My React Portfolio**
A personal portfolio website built with React to showcase my projects, skills, and experience as a web developer.
About
This portfolio highlights my work, technical skills, and achievements. It’s fully responsive, modern, and easy to navigate. Perfect for sharing with recruiters, clients, or collaborators.

**Features**
Personal introduction
Skills section 
Projects section 
Contact 

Responsive design for desktop, tablet, and mobile
Smooth scrolling and interactive UI

**Technologies Used**
1. React – For building user interfaces
2. React Router – Page navigation
3. Tailwind CSS  – Styling and layout
4. Vite – Project setup and development


**Installation**


1. Clone the repository:
git clone https://github.com/Anshikam0905/Portfolio
2. cd portfolio
3. Install dependencies:
  npm install
4. Start the development server:
npm start

The app will run at http://localhost:3000.
